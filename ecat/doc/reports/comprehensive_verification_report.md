## 3. SPI从机接口设计

### 3.1 设计规格

**接口标准**: SPI Mode 0/3 (CPOL=0, CPHA=0 或 CPOL=1, CPHA=1)

| 参数 | 规格 | 说明 |
|------|------|------|
| **时钟频率** | 最高40MHz | 与LAN9252对等 |
| **数据宽度** | 8位 | 标准SPI |
| **传输模式** | 全双工 | 同时收发 |
| **片选** | 低电平有效 | 标准CS# |
| **字节序** | MSB先行 | Big-Endian |

### 3.2 寄存器访问协议

#### 命令格式
```
[CMD:8bit][ADDR_H:8bit][ADDR_L:8bit][DATA0:8bit]...[DATA_N:8bit]
```

**命令码定义**:
- `0x02`: 读命令 (Fast Read)
- `0x03`: 读命令 (Normal Read) 
- `0x04`: 写命令 (Normal Write)
- `0x06`: 写命令 (Fast Write)

#### 读取时序
```
CS#  ────┐____________________┌────
CLK  ____┌┐┌┐┌┐┌┐┌┐┌┐┌┐┌┐┌┐┌┐____
MOSI ════╬CMD╬ADDR_H╬ADDR_L╬════
MISO ════════════════╬DATA0╬DATA1╬
```

#### 写入时序
```
CS#  ────┐________________________┌────
CLK  ____┌┐┌┐┌┐┌┐┌┐┌┐┌┐┌┐┌┐┌┐┌┐┌┐____
MOSI ════╬CMD╬ADDR_H╬ADDR_L╬DATA0╬DATA1╬
MISO ════════════════════════════════════
```

### 3.3 实现架构

```
┌─────────────────────────────────────┐
│  ecat_pdi_spi.sv (新增)             │
│  ┌──────────────┐  ┌──────────────┐│
│  │ SPI接口逻辑  │→│ 命令解析器   ││
│  │ - 状态机     │  │ - 读/写命令 ││
│  │ - 时钟采样   │  │ - 地址解码  ││
│  │ - 数据移位   │  │ - 数据缓冲  ││
│  └──────────────┘  └──────────────┘│
│           ↓                ↓        │
│  ┌──────────────────────────────┐  │
│  │   内部总线桥接               │  │
│  │   - 转换为Avalon格式        │  │
│  │   - 仲裁SPI/Avalon访问      │  │
│  └──────────────────────────────┘  │
│           ↓                         │
└───────────────────────────────────┘
            │
            ↓
    ecat_register_map.sv
```

### 3.4 实现文件

**新增文件**: `rtl/interface/ecat_pdi_spi.sv` (~350行)

**关键模块**:
```systemverilog
module ecat_pdi_spi #(
    parameter CLK_FREQ_HZ = 50_000_000,
    parameter MAX_SPI_FREQ = 40_000_000
)(
    // System
    input  wire         clk,
    input  wire         rst_n,
    
    // SPI Interface
    input  wire         spi_sck,
    input  wire         spi_cs_n,
    input  wire         spi_mosi,
    output reg          spi_miso,
    
    // Internal Bus (to register_map)
    output reg          bus_req,
    output reg          bus_wr,
    output reg [15:0]   bus_addr,
    output reg [31:0]   bus_wdata,
    input  wire [31:0]  bus_rdata,
    input  wire         bus_ack
);
```

**状态机**:
- `IDLE`: 等待CS#拉低
- `CMD`: 接收命令字节
- `ADDR_H`: 接收地址高字节
- `ADDR_L`: 接收地址低字节
- `DATA`: 传输数据字节
- `WAIT_ACK`: 等待内部总线响应
